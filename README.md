## Requirements

* iOS 8.0+ 
* Xcode 7.3 or above

## Installation

### CocoaPods
####1.在你的项目Podfile里面添加下面的内容

source 'https://github.com/CocoaPods/Specs.git'

platform :ios, '8.0'
use_frameworks!

pod 'UsefulPickerView', '~> 0.0.9'

###2.终端中执行命令 pod install
###3. 使用{Project}.xcworkspace打开项目


---
###或者直接下载,将下载文件的Scource文件夹下的文件拖进您的项目中就可以使用了
